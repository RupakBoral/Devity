# 🌐 Devity – Developer Collaboration Platform

**Devity** is an online social platform where developers can share ideas, showcase projects, and collaborate to build impactful solutions together. Connect with like-minded individuals, brainstorm, and innovate in a community that thrives on creativity and teamwork.

> *Meet developers who share your passion. Collaborate, contribute, and create something big.*


## 🚀 Features

* 🔐 **Authentication & Authorization** – Secure login system to protect user data
* 🙋‍♂️ **Personalized Profiles** – Customize your developer profile and showcase your work
* 📂 **Project Showcase** – Explore and display innovative projects from the community
* 🔗 **Networking** – Connect with developers across domains and expand your network
* 💬 **Live Chat** – Exchange ideas and collaborate in real-time with peers
* 🌗 **Light & Dark Mode** – Switch between themes for a better visual experience


## 🛠️ Tech Stack

### **Frontend**

* React.js
* Redux / Context API
* Tailwind CSS
* Axios
* Daisy UI

### **Backend**

* Node.js
* Express.js
* MongoDB
* JWT for authentication

### **Other Tools**

* Socket.io (for real-time chat)
* Mongoose
* Git & GitHub
